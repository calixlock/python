- python 가상환경 생성

```shell
python -m venv [projectName]
```

- python 가상환경 실행 / windows 10 env

```shell
[projectName]\Scripts\activate
```

- python 가상환경 특정 버젼 설치

```shell
pip install django==5.0.3
```

- python 가상환경 종료

```python
deactivate
```
